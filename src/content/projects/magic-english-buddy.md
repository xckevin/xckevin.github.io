`Magic English Buddy` 是一个面向儿童英语启蒙的学习产品，定位为“离线优先的英语学习伙伴（Companion）”，通过沉浸式世界观与游戏化机制提升学习持续性。

## 核心价值

- 离线优先：核心学习链路（读、查、练、存）可离线运行
- 轻量普适：兼顾低端设备可用性，降低接入门槛
- 兴趣驱动：通过成长系统与关卡解锁，减少枯燥重复训练
- 反馈闭环：二维码同步学习数据，支持家长/老师跟踪进度

## 主要功能

- 七级分级地图（L1~L7）：基于 i+1 理论逐级推进内容难度
- 魔法镜阅读器：TTS 跟读高亮、影子跟读、离线词典查词
- Buddy 养成系统：通过学习行为积累“魔力值”推动角色成长
- 魔法挑战 Quiz：分层题型设计，强化提取练习与长期记忆
- 守护者卷轴：支持学习报告可视化与证书导出

## 技术栈

- React 19 + TypeScript + Vite 6
- Zustand + XState
- Dexie.js（IndexedDB）
- Framer Motion + Lottie
- vite-plugin-pwa + Workbox
- CSS Modules + CSS Variables
